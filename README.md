# Tweeter Project

Tweeter is a simple, single-page Twitter clone.


## Dependencies

- Express
- Node 5.10.x or above
- body-parser
- chance
- dotenv
- handlebars
- md5
- moment
- mongodb

## Installation

1. npm install
2. create .env file with path /tweeter
3. Enjoy :)
